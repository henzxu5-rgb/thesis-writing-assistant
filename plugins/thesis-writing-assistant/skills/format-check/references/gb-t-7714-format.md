# GB/T 7714-2015 参考文献著录规则

## 文献类型标识

| 类型 | 标识 | 示例 |
|------|------|------|
| 普通图书 | [M] | 专著、教材 |
| 期刊文章 | [J] | 学术论文 |
| 学位论文 | [D] | 博士/硕士论文 |
| 会议论文 | [C] | 学术会议 |
| 报纸文章 | [N] | 报刊评论 |
| 报告 | [R] | 研究报告 |
| 标准 | [S] | 国家标准 |
| 电子资源 | [EB/OL] | 网络资源 |
| 汇编 | [G] | 论文集 |
| 析出文献 | [A] | 论文集中的单篇 |

## 著录格式

### 专著 [M]

```
著者. 书名[M]. 版本（第1版不标注）. 出版地: 出版者, 出版年.
```

示例（不含页码——引用整本书或来源无页码标记时）：
- 康德. 纯粹理性批判[M]. 邓晓芒, 译. 北京: 人民出版社, 2004.
- 康德. 法的形而上学原理[M]. 沈叔平, 译. 北京: 商务印书馆, 2012.
- 张文显. 法哲学范畴研究[M]. 修订版. 北京: 中国政法大学出版社, 2001.

示例（含页码——引用特定段落且来源有页码标记时）：
- 康德. 道德形而上学（注释本）[M]. 张荣, 李秋零, 译注. 北京: 中国人民大学出版社, 2013: 230.
- 康德. 道德形而上学的奠基[M]. 李秋零, 译. 北京: 中国人民大学出版社, 2013: 387-388.

### 译著 [M]

```
原著者. 书名[M]. 译者, 译. 出版地: 出版者, 出版年.
```

示例：
- [德]拉德布鲁赫. 法哲学[M]. 刘幸义, 等译. 北京: 法律出版社, 2005.
- [德]哈贝马斯. 在事实与规范之间[M]. 童世骏, 译. 北京: 三联书店, 2003.

注意：外国著者姓名前加方括号标注国籍。

### 期刊论文 [J]

```
著者. 篇名[J]. 刊名, 年, 卷(期): 起止页码.
```

示例：
- 舒国滢. 法哲学沉思录[J]. 比较法研究, 2019, 33(5): 1-15.
- 郑永流. 康德的权利概念[J]. 法学研究, 2020(3): 15-28.

### 学位论文 [D]

```
著者. 题名[D]. 保存地: 保存单位, 年份.
```

示例：
- 李某某. 康德法权思想研究[D]. 北京: 中国政法大学, 2019.

### 论文集中的析出文献 [A]

```
析出文献著者. 析出文献题名[A]. 论文集著者. 论文集题名[C]. 出版地: 出版者, 年: 页码.
```

### 电子资源 [EB/OL]

```
著者. 题名[EB/OL]. (发布日期)[引用日期]. 网址.
```

### 外文文献

外文文献按照同样的格式规则著录，保持原文语言：
- Ripstein A. Force and Freedom: Kant's Legal and Political Philosophy[M]. Cambridge: Harvard University Press, 2009.
- Höffe O. Kant's Cosmopolitan Theory of Law and Peace[M]. Cambridge: Cambridge University Press, 2006.

注意：外文著者姓在前，名缩写在后。

## 文内引用格式（脚注制）

- 文内用方括号上标标注：`文本内容[1]`
- 每一处引用分配一个独立编号，编号按在文中出现的顺序递增
- **同一文献多次引用时，每次引用使用不同编号**，各条脚注标注各自的具体页码
- 同一处同时引用多篇文献：`[1][2]`（分别标注，不合并）
- **不要**在正文行内标注科学院版页码（如 `(6:225)`），页码一律写在脚注中

示例：
```
正文：……它的简单性……当然一开始就必定使人惊奇。[1]
正文：……都被康德称为"自由的法则"[2]

[1] 康德. 道德形而上学（注释本）[M]. 张荣, 李秋零, 译注. 北京: 中国人民大学出版社, 2013: 225.
[2] 康德. 道德形而上学（注释本）[M]. 张荣, 李秋零, 译注. 北京: 中国人民大学出版社, 2013: 214.
```

## 注意事项

- 同一文献在文中多次引用不同页码时，使用不同编号，每条脚注标注各自页码
- 转引需注明"转引自"
- 中文文献在前，外文文献在后
- 同一语言文献按著者姓氏首字母/笔画排序
- 必填项缺失时不可编造，应标注 `[待补充]`
