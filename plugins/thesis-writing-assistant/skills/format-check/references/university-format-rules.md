## 已确定的格式要求

### 章节编号
- 统一使用阿拉伯数字编号：`1` → `1.1` → `1.1.1`
- 不使用"第一章""一、""（一）"等中文序号
- Markdown 对应：`# 1　标题` / `## 1.1　标题` / `### 1.1.1　标题`
