# 法学论文大纲模板参考

## 常见结构模式

### 模式一：经典三段式（适合理论阐释型）
1. 绪论（研究背景、问题提出、文献综述、研究方法）
2. 理论阐释（核心概念与理论体系）
3. 现实分析（理论的现代展开或实践考察）
4. 结论

### 模式二：比较研究型
1. 绪论
2. A 的理论/制度分析
3. B 的理论/制度分析
4. A 与 B 的比较分析（异同、传承、超越）
5. 对中国的启示或理论反思
6. 结论

### 模式三：问题导向型
1. 绪论（提出核心问题）
2. 问题的历史脉络
3. 现有理论的回应与不足
4. 新的分析框架或论证
5. 结论

### 模式四：历史—理论—实践三层递进
1. 绪论
2. 历史考察（概念的起源与演变）
3. 理论分析（核心理论体系）
4. 实践审视（理论的现代意义）
5. 结论

## 各部分写作要点

### 绪论
- 研究背景与问题意识（为什么研究这个问题）
- 国内外研究现状综述（前人做了什么，还有什么不足）
- 研究方法与创新点
- 论文结构安排
- 通常占全文 10-15%

### 正文各章
- 每章围绕一个核心论点展开
- 章与章之间有明确的逻辑过渡
- 每节有明确的论证目标
- 重要论断有文献支撑

### 结论
- 总结核心发现
- 回应绪论中提出的研究问题
- 指出研究局限性
- 展望后续研究方向
- 通常占全文 5-8%

## 法学论文特殊要求

- 概念界定要清晰：核心概念在正文开始前需严格定义
- 规范分析与实证分析相结合
- 注意区分"是什么"（描述性）与"应当是什么"（规范性）
- 引用法律条文时注明法律名称、条款编号、生效日期
- 外国法概念需保留原文术语并给出中文对应
