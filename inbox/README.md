# inbox — 原始文献入口

将你要导入的参考文献 MD 文件放在这个目录下，然后运行 `/process-reference`。

## 支持的文件格式

- `.md` 文件（Markdown 格式的文献全文或章节）

## 如何准备文献 MD 文件

### 整本书
- 可以将整本书转为一个大 MD 文件
- 也可以按章节分成多个 MD 文件（推荐，尤其是内容较多时）
- 建议保留原书的标题层级（`#` 章标题、`##` 节标题等）

### 单篇论文
- 每篇论文一个 MD 文件
- 包含论文的完整文本

### 建议在文件开头包含的信息
```markdown
# 文献标题

- 作者：
- 出版信息：（出版社/期刊名/年份等）
- 译者：（如为译著）

---

正文内容...
```

## 处理流程

运行 `/process-reference` 后：
1. AI 会分析文献结构并自动切分成小块
2. 每块约 500-1500 字，保持论证完整性
3. 切分后的文件保存到 `library/` 目录
4. 全局索引 `library/index.md` 会被更新
5. 原始文件会被移到 `inbox/processed/`
